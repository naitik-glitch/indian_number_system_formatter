# 🔢 Indian Number Formatter

## About

A simple Python program that converts an integer into the Indian number format by adding commas in the correct places.

## Features

- Takes an integer as input
- Formats the number using the Indian numbering system
- Checks for invalid input
- Allows the user to format multiple numbers

## Example

Input:
```
12345678
```

Output:
```
1,23,45,678
```

## Skills Used

- Functions
- Lists
- String Slicing
- Loops (`while`)
- User Input
- Conditional Statements
- Exception Handling (`try` and `except`)

## Run the Program

```bash
python indian_number_formatter.py
```
